## 💰 成本构成图

```
月度总成本：~$12-20/月

┌──────────────────────────────────────┐
│  计算资源 (Compute)     ~$5/月       │
│  ├─ 后端服务 (0.5 CPU)               │
│  └─ 前端服务 (0.25 CPU)              │
├──────────────────────────────────────┤
│  数据库 (Database)      ~$7/月       │
│  └─ PostgreSQL 基础版                 │
├──────────────────────────────────────┤
│  带宽 (Bandwidth)       $0-5/月      │
│  └─ 1-5GB/月（免费额度内）            │
├──────────────────────────────────────┤
│  域名 (Domain)          $0-15/年     │
│  └─ 自定义域名（可选）                │
└──────────────────────────────────────┘

💡 优化建议：
- 初期用 SQLite 可省$7/月
- 开启休眠模式节省 50%
- 1GB 免费带宽通常够用
```
